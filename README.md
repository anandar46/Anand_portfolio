# Personal Portfolio - Anand

A modern, responsive, and minimalist personal portfolio website built with HTML, CSS, and JavaScript. The site showcases my skills, experience, academic projects, and open-source contributions.

## Features

- **Premium Design:** Clean aesthetics, subtle glassmorphism, and smooth scroll animations.
- **Dark/Light Mode:** Integrated theme toggle with system preference detection.
- **Dynamic Background:** Animated, floating blurred blobs for a premium visual experience.
- **Live GitHub Integration:** Automatically fetches and displays recent public repositories from my GitHub profile (`anandar46`).
- **Responsive Layout:** fully responsive and works seamlessly across desktops, tablets, and mobile devices.

## Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, media queries, and animations.
- **JavaScript (Vanilla):** Theme toggling, Intersection Observer for scroll animations, and Fetch API for GitHub repositories.

## Setup & Local Development

This is a static website, so you don't need any build tools to run it locally.

1. Clone the repository or download the files.
2. Open `port.html` directly in your web browser.
3. *Optional:* To avoid CORS issues when testing the GitHub API locally, use a simple local server. For example, if you have Python installed, you can run:
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000/port.html` in your browser.
